Create `db/__init__.py` (empty) and `db/models.py` using SQLAlchemy 2.0 style — `DeclarativeBase` and `mapped_column`, NOT legacy `declarative_base()` or `Column()`.

The four models with EXACT columns (no omissions — these are the schema contracts):

**Prediction** (`predictions` table):
- `id`: UUID primary key, server_default=gen_random_uuid()
- `predicted_at`: TIMESTAMPTZ NOT NULL, server_default=now()
- `model_version`: String(50) NOT NULL
- `features`: JSON NOT NULL
- `score`: Float NOT NULL
- `decision`: String(10) NOT NULL — CHECK constraint: `decision IN ('approved', 'denied')`
- `path`: String(20) NOT NULL — CHECK constraint: `path IN ('model', 'checkerboard')` — CRITICAL: required by CB-PDD router in Phase 3
- `simulation_day`: Integer nullable=True — null for live predictions, integer for simulator-generated data

**Outcome** (`outcomes` table):
- `id`: UUID primary key, server_default=gen_random_uuid()
- `prediction_id`: UUID FK → predictions.id, unique=True, NOT NULL
- `actual_default`: Boolean NOT NULL
- `predicted_at`: TIMESTAMPTZ NOT NULL — copied from predictions.predicted_at at write time
- `outcome_received_at`: TIMESTAMPTZ NOT NULL — CRITICAL: separate from predicted_at; models label delay (Phase 4 SIM-04 reads this)

**DriftScore** (`drift_scores` table):
- `id`: UUID primary key, server_default=gen_random_uuid()
- `computed_at`: TIMESTAMPTZ NOT NULL, server_default=now()
- `drift_score`: Float NOT NULL
- `psi_score`: Float nullable=True — for PSI baseline comparison (Phase 4 DRIFT-05)
- `threshold_crossed`: Boolean NOT NULL
- `window_days`: Integer NOT NULL
- `trial_count`: Integer nullable=True

**Alert** (`alerts` table):
- `id`: UUID primary key, server_default=gen_random_uuid()
- `fired_at`: TIMESTAMPTZ NOT NULL, server_default=now()
- `drift_score`: Float NOT NULL
- `retrain_run_id`: String(100) nullable=True — null when retrain was skipped
- `promoted`: Boolean nullable=True — null when retrain was skipped

Add Google-style docstrings to each class. Add Python 3.11+ type hints throughout. No hardcoded connection strings anywhere in this file.
