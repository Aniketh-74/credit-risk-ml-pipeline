      (the next plausible maximum given quarterly credit reporting cycles). We apply clip(upper=17)."
    - Show before/after value_counts with the cap applied

    **Section 6 — Spearman Correlation Heatmap**
    - Compute Spearman correlation matrix (not Pearson) using df.corr(method='spearman')
    - Drop NaN rows before computing (use df.dropna())
    - Plot seaborn heatmap with annot=True, fmt='.2f', figsize=(12, 10)
    - Markdown cell: "Spearman rank correlation is used instead of Pearson because credit features
      are heavily right-skewed. Pearson assumes linear relationships and is sensitive to outliers —
      both assumptions are violated here."
    - Identify and state the strongest correlations (expected: NumberOfTime30-59, 60-89, 90DaysLate
      are highly correlated with each other ~0.98)

    **Section 7 — Class Imbalance Summary and SMOTE Rationale**
    - Markdown cell summarizing the problem and solution:
      "Summary: Give Me Credit has three preprocessing challenges that must be addressed before
      training: (1) 7% class imbalance — addressed with SMOTE applied only to the training split;
      (2) MonthlyIncome 19.8% NaN — addressed with median imputation fitted only on train, plus
      a missingness indicator flag; (3) NumberOfTimes90DaysLate outliers at 96/98 — addressed
      with clip(upper=17). SMOTE is applied AFTER the train/test split to prevent test set
      contamination — this is the only correct approach per imbalanced-learn official docs."
