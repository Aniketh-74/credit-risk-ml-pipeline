using the MLflow 3.x API. Then run a CB-PDD smoke test to validate that the model's score
distribution produces enough signal for the detector to fire on a simulated 30-day denial loop —
catching the class imbalance pitfall before Phase 4 builds orchestration around it.

success criterion in Phase 2) exists specifically to detect the failure mode where a poorly
calibrated model produces scores clustered near 0.05-0.10 — making CB-PDD blind to drift.
Testing at τ∈{500, 1000, 2000} gives directional guidance for Phase 4's parameter choice.

      - Creates `MlflowClient()`

      - Calls `client.get_run(mv.run_id)` to get the run
      - Returns `run.data.metrics.get("auc_test", 0.0)` as float
      - Wraps in try/except — returns 0.0 if no champion exists or any registry error
      - Google docstring with Returns section

    - `load_champion() -> object`
      - Returns `mlflow.pyfunc.load_model(f"models:/{MODEL_NAME}@champion")`
      - Google docstring: "Load the @champion model for inference. Returns MLflow pyfunc model."

       ```python
       def simple_cbpdd_smoke(records: list[dict], tau: int, alpha: float = 0.05) -> dict:
           """Minimal CB-PDD simulation — count trials and check for score distribution shift.

           This is a simplified version for Phase 2 validation only. The full CB-PDD
           implementation (CheckerBoardPredictor, DensityChangeTracker) is Phase 4's work.

           Args:
               records: List of prediction dicts with keys: simulation_day, score, decision.
               tau: Threshold for number of "checkerboard" trials before testing.
               alpha: Significance level for drift detection.

           Returns:
               Dict with keys: tau, detections (int), detection_days (list[int]).
           """
           # Simplified: track rolling mean score per day; detect shift via Mann-Whitney
           from scipy import stats

           days = sorted(set(r["simulation_day"] for r in records))
           if len(days) < 14:
               return {"tau": tau, "detections": 0, "detection_days": []}

           # Use first 7 days as reference window
           ref_scores = [r["score"] for r in records if r["simulation_day"] < 7]
           detections = 0
           detection_days = []
           trial_count = 0

           for day in days[7:]:
               day_scores = [r["score"] for r in records if r["simulation_day"] == day]
               trial_count += len(day_scores)
               if trial_count >= tau:
                   _, p_value = stats.mannwhitneyu(ref_scores, day_scores, alternative="two-sided")
                   if p_value < alpha:
                       detections += 1
                       detection_days.append(day)
                   trial_count = 0  # reset after each test

           return {"tau": tau, "detections": detections, "detection_days": detection_days}
       ```

       ```
       τ=2000: detections=X, first detection at day Y
       ```
